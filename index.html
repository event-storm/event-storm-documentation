Getting started

Easy to Integrate

Event Storm is designed from the ground up to be an easy replacement for your state management choice. Already familiar API with an optimized, decentralized subscription mechanism

Performance Centric

Each feature of Event Storm is desgined with the performance and scale in mind. Event Storm will allow you to minimize the application rendering to the rare minimum. Bundle size, integration cost, subscription mechanism, recomputions, etc.

UI Agnostic

The Event Storm does not rely on any UI framework/library. You can use the Event Storm with any UI solution on your target platform(browser, backend and native)
